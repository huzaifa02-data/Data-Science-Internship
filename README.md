### **Task 1: Term Deposit Subscription Prediction**

**Objective:**
Predict whether a bank customer will subscribe to a term deposit.

**Key Highlights:**

* Data cleaning and categorical encoding
* Logistic Regression & Random Forest models
* Evaluation using Confusion Matrix, F1-Score, and ROC Curve
* Model explainability using **SHAP/LIME**

**Skills Gained:**

* Classification modeling
* Feature engineering
* Explainable AI (XAI)
* Customer behavior analysis
  
### **Task 2: Customer Segmentation Using Unsupervised Learning**

**Objective:**
Cluster customers based on spending behavior and recommend marketing strategies.

**Key Highlights:**

* Exploratory Data Analysis (EDA)
* K-Means clustering
* Cluster visualization using PCA / t-SNE
* Data-driven marketing strategy recommendations

**Skills Gained:**

* Unsupervised learning
* Dimensionality reduction
* Customer segmentation
* Business insight generation

---

### **Task 3: Energy Consumption Time Series Forecasting**

**Objective:**
Forecast household energy usage using historical data.

**Key Highlights:**

* Time series parsing and resampling
* Feature engineering (hour, weekday, weekend)
* Model comparison: ARIMA, Prophet, XGBoost
* Visualization of actual vs predicted values

**Skills Gained:**

* Time series forecasting
* Feature engineering
* Model evaluation (MAE, RMSE)
* Temporal data visualization.

### **Task 4: Loan Default Risk with Business Cost Optimization**

**Objective:**
Predict loan default risk and optimize decisions using cost-based thresholds.

**Key Highlights:**

* Binary classification models
* Business cost definition for FP & FN
* Threshold tuning to minimize business loss
* Feature importance analysis

**Skills Gained:**

* Risk modeling
* Cost-sensitive learning
* Business-driven ML decisions
* Model interpretability

### **Task 5: Interactive Business Dashboard (Streamlit)**

**Objective:**
Build an interactive BI dashboard for sales and profit analysis.

**Key Highlights:**

* Data preprocessing and aggregation
* Interactive filters (Region, Category, Sub-Category)
* KPI visualization:

  * Total Sales
  * Total Profit
  * Top 5 Customers by Sales

**Skills Gained:**

* Business Intelligence (BI)
* Streamlit dashboard development
* Data storytelling
* User interactivity
